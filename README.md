# Visualizing K-Means Clustering in Iris Dataset
This simple Shiny App illustrates K-means clustering in the iris dataset. You can customize a variety of parameters for both the clustering algorithm and the outputted visualization using the widgets baked into the app.

This app is hosted on Shinyapps.io, [click here to see it in action!](https://cgj-tang.shinyapps.io/kmeans_shiny/)
